# ListenBrainz Dashboard Plugin for Nuclear

A dashboard provider plugin that surfaces sitewide charts, trending artists, top albums, and new releases from [ListenBrainz](https://listenbrainz.org/).

## Features

- Top tracks, artists, and albums from ListenBrainz sitewide statistics
- Fresh releases from the past 30 days
- Cover art from the [Cover Art Archive](https://coverartarchive.org/)
- Links to the [MusicBrainz metadata provider](https://github.com/NuclearPlayer/nuclear-plugin-musicbrainz) for artist and album details

## Limitations

- No editorial playlists. ListenBrainz doesn't offer public playlists without a user account
- No artist images. ListenBrainz and MusicBrainz don't serve artist photos

## Installation

Install from the Nuclear plugin browser.
